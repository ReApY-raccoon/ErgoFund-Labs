# Contributing to ErgoFund-Labs

Thanks for your interest in the Bene Fundraising Platform (AOSSIE Technologies).

## How to contribute

1. Fork the repo: [github.com/Aahilmak786/ErgoFund-Labs](https://github.com/Aahilmak786/ErgoFund-Labs)
2. Create a branch from `main`, make small commits (see [ROADMAP.md](./ROADMAP.md) for commit habits).
3. Run `npm run lint`, `npm run format:check`, `npm test`, `npm run build`.
4. Open a Pull Request against `main`.

## Code style

- ESLint and Prettier are enforced in CI. Run `npm run lint` and `npm run format` locally.
